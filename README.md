# Clinical Trials Classifier

## Version 1.0.0 (Alpha)

### By Skye Lane Goetz And Gwênlyn Glusman

Improved DuckDB Export Of The NCATS Translator BABEL Project With Enhanced Indexing and NLP Capabilities. Dbssert Is Intended To Be Used With The Tablassert Knowledge Graph Backend's Fullmap Feature.

## Usage (With Nix)

Develop Nix Shell
```bash
nix develop -L .
```

Use Exposed CLI
```bash
dbssert-cli --help
```
